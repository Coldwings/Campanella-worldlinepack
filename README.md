# Campanella-worldlinepack
Campanella的前端界面

项目依赖
---
- 使用[npm](https://github.com/npm/npm)
  或者[npm](https://github.com/npm/npm)管理js包。
- 使用[webpack](https://github.com/webpack/webpack)打包js。


安装依赖
---
```shell
npm install  # or, cnpm install
```

Dev-Server
---
```shell
npm start
```
访问 [localhost:3000](localhost:3000)

编译压缩打包
---
```shell
npm run dist
```

语法检查
---
```shell
npm run lint
```
